# **App Name**: Pandora-Intelligence-System

## Core Features:

- Mobile-First Layout: Responsive layout optimized for mobile devices (Samsung A16 screen size).
- Bottom Navigation Bar: Implement a bottom navigation bar with four sections: Dossiers, Market, Ops, and Network.
- Mock Dashboard: Create a dashboard page displaying a list of 'Assets' with Green/Red status badges.
- Firebase Hosting Configuration: Configure the `firebase.json` file for deployment to Firebase App Hosting.
- Gemini Integration: Enable Gemini LLM

## Style Guidelines:

- Background color: True Black (#000000) to provide a dark, terminal-like aesthetic.
- Primary color: Electric Purple (#BF04BF) for contrast against the black background, reflecting tech and security.
- Accent color: Neon Green (#39FF14) for status badges and interactive elements, inspired by a classic command-line interface.
- Font: 'Source Code Pro' (monospace) for a 'Terminal' look and feel.
- Mobile-first design with a focus on clear, concise information display.
- Simple, geometric icons for navigation and asset status.
